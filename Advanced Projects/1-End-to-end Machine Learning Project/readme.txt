There are two python files that you can run with streamlit to see thses apps as a web application.
to run this apps you should install streamlit with pip. it's like this: "pip install streamlit".

after installation you can run those files with this command: "streamlit run ml.py" or "streamlit run ml_2.py"

Additionally, there is a comprehensive, step-by-step Jupyter notebook wherein we develop a program for the purpose of detecting emotions within text.
